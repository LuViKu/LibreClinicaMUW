A small, opinionated system for the LibreClinica MUW cCRF platform — built for the Department of Ophthalmology and Optometry at the Medical University of Vienna. Rooted in the MedUni Wien Corporate Design, sharpened for data-dense clinical workflows.

Principle 01

Clinical, not corporate.

Data density first. Status carries text AND colour. Every error explains itself. The chrome stays quiet so the data leads.

Principle 02

MUW where it matters.

Dunkelblau anchors brand moments — wordmark, primary action, focus ring. Coral is the warm accent. Secondary colors map to roles.

Principle 03

Audited by default.

Every regulated action shows what is recorded, by whom, when. Reversibility and consequences live in the affordance, not in the help docs.

For developers

Use the system in any mockup.

Three lines in the head and the entire MUW palette + Newsreader / Inter pairing is available as Tailwind utility classes. No build step.

<script src="https://cdn.tailwindcss.com"></script>
<script src="muw-tailwind-config.js"></script>
<link rel="stylesheet" href="muw-tokens.css" />

class="bg-muw-blue"
class="text-muw-teal-700"
class="muw-display text-3xl"
class="rounded-muw shadow-muw-card"
